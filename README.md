# Das Kompendium

Dieses Projekt ist ein digitales Nachschlagewerk und Tracking-Tool für TableTop Roleplay Games (TTRPGs). Es bietet die Möglichkeit, die im Laufe der Kampagne entstandenen Informationen strukturiert und hoch anpassbar zu erstellen und zu verwalten.

## Hauptfunktionen

- Übersichtliche Darstellung von generiertem Wissen und Welten
- Schnelle Such- und Filtermöglichkeiten
- Strukturierung von Daten
- Tool zur einfachen Dokumentation und Nachverfolgung aller Entwicklungen einer Kampagne

## Zielgruppe

Spieler und Spielleiter, die ihre TTRPG-Kampagnen strukturieren und digitalisieren wollen.

## Technologie

Das Projekt ist im Framework von next.js entwickelt. Die Datenbank basiert auf PostGreSQL und wird von supabase gehostet.
